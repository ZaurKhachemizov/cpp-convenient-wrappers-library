Unsubscribes listener from the event.

Template Parameters
_EventA type of event that listener is unsubscribing.
_Callable[1] A type of function object or function.
_Function[2] - [3] A type of member function.
_Object[2] - [3] A class object whose _Function is a member.
Parameters
[in]_callable[1] A reference to function object or pointer/reference to a function that was subscribed earlier.
[in]_function[2] - [3] A pointer to a member function that was subscribed earlier.
[in]_object[2] A constant reference to object of std::shared_ptr<_Object> type.
[3] A constant reference to object of boost::shared_ptr<_Object> type.
Returns
No return value.
Complexity
Linear in the number of listeners subscribed to the event.
Exception safety
Will not throw unless a user destructor or equality operator == throws. If either throw, the listener may stay not removed.
