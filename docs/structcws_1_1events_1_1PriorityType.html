Detailed Description

template<typename _Priority = int, typename _Comparator = std::less<_Priority>>
struct cws::events::PriorityType< _Priority, _Comparator >

Specifies type of priority and method for determining higher priority.

Uses as a template parameter of Dispatcher class and dispatcher::Type structure.

Template Parameters
_PriorityA type of priority that is used for specifying listeners' invocation order.
_ComparatorA type of method used for determining higher priority.
Remarks
Invocation order:
1) Listeners subscribed without specifying priority value and with _order = Order::FRONT.
2) Listeners subscribed with a particular priority according to the ordering of their priority values.
3) Listeners subscribed without specifying priority value and with _order = Order::BACK.
