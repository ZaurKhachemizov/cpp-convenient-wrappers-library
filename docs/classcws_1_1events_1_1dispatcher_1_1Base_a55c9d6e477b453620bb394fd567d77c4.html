void cws::events::dispatcher::Base< _Mutex, _Priority, _Comparator, _Events >::remove_listeners ( )
inline

Unsubscribes several listeners.

[1] Unsubscribes all listeners from the event with the specified priority.
[2] Unsubscribes all listeners from all events.

Template Parameters
_Event[1] A type of event that listeners are unsubscribing.
Parameters
[in]_priority[1] Listeners with what priority value must be unsubscribed.
Returns
No return value.
Complexity
[1] Logarithmic in the number of priority values plus the number of listeners subscribed with the specified priority.
[2] Linear in the number of listeners of all events.
Exception safety
[1] Will not throw unless equality operator == throws.
[1] - [2] Will not throw unless a user destructor throws.
If either throw, some listeners may stay not removed.
