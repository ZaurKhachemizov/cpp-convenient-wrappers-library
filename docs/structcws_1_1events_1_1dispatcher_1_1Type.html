Detailed Description

template<typename ... _Types>
struct cws::events::dispatcher::Type< _Types >

Specifies custom Dispatcher type.

A convenient way to declare Dispatcher of custom type with specified mutex type and/or priority type and events list.

Template Parameters
..._TypesCan contain MutexType and/or PriorityType. Must contain TypesList.
