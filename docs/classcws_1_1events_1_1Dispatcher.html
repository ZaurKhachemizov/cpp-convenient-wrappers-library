Detailed Description

template<typename ... _Events>
class cws::events::Dispatcher< _Events >

Default Dispatcher class.

Dispatcher class is used to subscribe listeners to events, unsubscribe listeners from events, and dispatch events to subscribed listeners.

Template Parameters
..._EventsTypes of events that dispatcher can dispatch.
Remarks
Dispatcher class is default-constructible, non-copyable, moveable.
By default, Dispatcher class is non-thread-safe.
By default, listeners' priority is of type int, and the method to determine the highest priority is std::less<int>.
